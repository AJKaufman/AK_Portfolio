PawPals

PawPals is a fun app that uses a backend repository to store user credentials in order to create profiles and collections of objects called "Cats".

You can change attributes on those pets after they are created, log off and back on, and still see those changes reflected later on.

Lessons Learned

Working on PawPals taught me about using repositories and gave me experience with making calls to the backend, using Redis. It also allowed me to evaluate user experience and helped me understand what makes a useful page useful and when to remove unnecessary content.
